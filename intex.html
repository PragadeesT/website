Line Following Robot
Objectives:
To create a robot that can follow a black line accurately on a surface, using sensor-based guidance and motorized movement. The robot achieved 95% accuracy in line tracking over a distance of 100 meters.
Steps in the project:
Image Processing and Classification:
  • Pre-processing: Cropping, resizing, normalization, and augmentation of the cell images to enhance model robustness
  • Feature extraction: Using convolutional layers in the model to identify important visual patterns, such as cell shapes or anomalies indicative of cancer.
Machine Learning Algorithms:
  • Python was used for implementation with libraries like TensorFlow and Keras.
  • A Convolutional Neural Network (CNN) was the primary model architecture. CNNs are particularly effective for image classification tasks due to their ability to learn spatial hierarchies from data.
Evaluation Metrics:
  • Precision: Measures how many of the identified positive cases were truly positive (low false positive rate).
  • Recall: Assesses how well the model identifies all true positive cases (low false negative rate).
  • F1-Score: The harmonic mean of precision and recall, giving a balanced view of the model's performance.
Applications:
This model provides a tool to assist medical professionals in diagnosing white blood cell cancers, potentially reducing manual analysis time and improving early detection rates.The project highlights the potential of machine learning to revolutionize healthcare by providing scalable, efficient diagnostic solutions.
